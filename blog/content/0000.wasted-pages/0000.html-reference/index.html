文档类型声明

返回索引

实例

                    
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <meta charset="utf-8"> 
                            <title>文档标题</title>
                        </head>
                        <body>
                            <p>文章内容</p>
                        </body>
                        </html>

运行效果：

文档标题

文章内容

浏览器支持

所有主流浏览器都支持该声明。

定义及使用说明

在HTML中，<!DOCTYPE>声明是DTD（文档类型声明，Document Type Declaration）的缩写，用于指示浏览器当前页面所使用的HTML版本。
<!DOCTYPE>声明位于文档中的最前面的位置，处于<html>标签之前，它不是一个HTML标签，因此不需要关闭。
在HTML 4.01及之前版本中，<!DOCTYPE>声明可能会指向DTD（文档类型定义，Document Type Definition，下同）文件，它定义了文档的结构和元素规则。因为HTML 4.01基于SGML（标准通用标记语言，Standard Generalized Markup Language）。DTD指定了标记语言的规则，确保了浏览器能够正确的渲染内容。
由于HTML5不基于SGML，因此不再依赖DTD文件，而是使用更简单的声明。
提示：总是为HTML文档添加<!DOCTYPE>声明，确保浏览器能够预先知道文档类型。

HTML 4.01与HTML5之间的差异

HTML 4.01规定了三种不同的<!DOCTYPE>声明，分别是：Strict、Transitional 和 Frameset。
而HTML5中仅规定了一种，即<!DOCTYPE html>。

注意事项

它没有结束标签，也不区分大小写。

常见声明

HTML5

                    <!DOCTYPE html>

HTML 4.01 Strict

                    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML 4.01 Transitional

                    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

HTML 4.01 Frameset

                    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

超文本链接标签

返回索引

实例

                    
                        <div id="target"><p>定位到这里 - div id="target"</p></div>
                        <p><a href="https://courageluo.github.io">网站链接 - courageluo.github.io</a></p>
                        <p><a href="../../../index.html">站内链接 - ../../../index.html</a></p>
                        <p><a href="../../../index.html" target="_blank">在新窗口打开链接 - target="_blank"</a></p>
                        <p><a href="mailto:someone@example.com?Subject=Hello%20again" target="_top">Email链接 - 发送“Hello again”</a></p>
                        <p><a href="#target" class="scroll-link">锚点链接 - #target</a></p>

运行效果：

定位到这里 - div id="target"

网站链接 - courageluo.github.io

站内链接 - ../../../index.html

在新窗口打开链接 - target="_blank"

Email链接 - 发送“Hello again”

锚点链接 - #target

浏览器支持

所有主流浏览器都支持该标签。

标签定义及使用说明

<a>标签定义超链接，用于从一个页面链接到另一个页面。
<a>元素最重要的属性是href属性，它指定链接的目标。
在所有浏览器中，链接的默认外观如下：

  • 未被访问的链接：带有下划线，蓝色
  • 已被访问的链接：带有下划线，紫色
  • 活动链接：带有下划线，红色

注意事项

如果没有使用href属性，则不能使用hreflang、media、rel、target以及type属性。
通常在当前浏览器窗口中显示被链接页面，除非规定了其他target。
使用CSS来改变链接的样式。

HTML 4.01与HTML5之间的差异

在HTML 4.01中，<a>标签既可以是超链接，也可以是锚。在HTML5中，该标签是超链接，但是假如没有href属性，它仅仅是超链接的一个占位符。
HTML5有一些新的属性，同时不再支持一些HTML 4.01的属性。
