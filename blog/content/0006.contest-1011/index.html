Problem A 互质

[题目描述]

给你一个正整数n，请问有多少个比n小的且与n互质的正整数？
两个整数互质的意思是，这两个整数没有比1大的公约数。

[输入]

输入包含多组测试数据。每组输入是一个正整数n（n≤1000000000）。当n=0时，输入结束。

[输出]

对于每组输入，输出比n小的且与n互质的正整数个数。

[解答]

                        
                            #include<iostream>
                            int main() {
                                int N,n;
                                while (scanf("%d",&N)!=EOF) {
                                    if (N==0) break;
                                    n=N;
                                    for (int i=2;i<=sqrt(N);i++) {
                                        if(N%i==0) {
                                            n=n/i*(i-1);
                                            while (N%i==0) N/=i;
                                        }
                                    }
                                    if (N>1) n=n/N*(N-1);
                                    std::cout<<n<<'\n';
                                }
                                return 0;
                            }

Problem B 美丽数

[题目描述]

小明很喜欢3和5这两个数字，他将能被3或5整除的数叫做美丽数。现在给你一个整数N（1≤N≤100000），你能告诉小明第N个美丽数是多少吗？

[输入]

输入包含多组测试数据。每组输入一个整数N（1≤N≤100000）。

[输出]

对于每组输入，输出第N个美丽数。

[解答]

由于100000并不大，所以可以求出前100000个美丽数，然后根据需求输出第N个美丽数即可。 

                        
                            #include<iostream>
                            #include<vector>
                            int main() {
                                int i=0;
                                std::vector<int> nums;
                                import:
                                    if (i%3==0||i%5==0) nums.push_back(i);
                                    i++;
                                if (nums.size()<=100000) goto import;
                                int a;
                                while (scanf("%d",&a)!=EOF) {
                                    std::cout<<nums[a]<<'\n';
                                }
                                return 0;
                            }
这里的goto语句就是一个循环，以上循环等价于 
                        
                            while (nums.size()<=100000) {
                                if (i%3==0||i%5==0) nums.push_back(i);
                                i++;
                            }

Problem C 开灯关灯

[题目描述]

有n盏灯，编号为1~n，第1个人把所有灯打开，第2个人按下所有编号为2的倍数的开关（这些灯将被关掉），第3个人按下所有编号为3的倍数的开关（其中关掉的灯将被打开，开着的灯将被关闭），以此类推。一共有k个人，问最后有哪些灯开着？

[输入]

输入数据有多组，输入n盏灯以及k（k≤n≤1000）。

[输出]

输出开着的灯的编号，参照样例输出。

[样例输入]

                        
                            7 3
                            8 2

[样例输出]

                        
                            1
                            5
                            6
                            7
                            1
                            3
                            5
                            7

[解答]

只需要用循环模拟这个开关灯过程就可以了。 

                        
                            #include<iostream>
                            int main() {
                                int n,k,m=1;
                                while (scanf("%d%d",&n,&k)!=EOF) {
                                    for (int i=1;i<=n;i++) {
                                        for (int p=2;p<=k;p++)
                                            if (i%p==0) m=!m;
                                        if (m) std::cout<<i<<'\n';
                                        m=1;
                                    }
                                }
                                return 0;
                            }

Problem D 大数取模

[题目描述]

现给你两个正整数A和B，请你计算$ A \: \mathrm{mod} \: B $。
为了使问题简单，保证B小于100000。

[输入]

输入包含多组测试数据。每行输入包含两个正整数A和B。A的长度不超过1000，并且0<B<100000。

[输出]

对于每一个测试样例，输出$ A \: \mathrm{mod} \: B $。

[解答]

经典的高精度算法。 

                        
                            #include<iostream>
                            #include<algorithm>

                            int mod(std::string a,int b) {
                                int d=0;
                                for (int i=0;i<a.size();i++)
                                    d=(d*10+(a[i]-'0'))%b;
                                return d;
                            }

                            int main() {
                                std::string a;
                                int b;
                                while (std::cin>>a>>b)
                                    std::cout<<mod(a,b)<<'\n';
                                return 0;
                            }

Problem E 同余问题

[题目描述]

给n个数，现在想知道有多少个大于1的整数被这n个数除得到的余数相同，请你帮他~

[输入]

第一个数为T，表示case数，每个case第一行输入一个2≤n≤100，表示数字个数。之后输入n个数，每个数都小于10^9。

[输出]

每个case输出一个数，表示大于1且满足除这n个已给出的数结果余数相同的数的个数。

[提示]

第一个样例三个数除以2都余0，除以4都余2，所以符合条件的有两个数。

[解答]

对于每数据逐一判断即可。 

                        
                            #include<iostream>
                            #include<algorithm>
                            
                            int a[101],b[101];
                            
                            int gcd(int a, int b) {
                                return b ? gcd(b,a%b) : a;
                            }
                            
                            int res(int n) {
                                int res=0;
                                for (int i=2;i*i<=n;i++) {
                                    if (n%i==0) {
                                        if (i*i!=n) res++;
                                        res++;
                                    }
                                }
                                if (n!=1) res++;
                                return res;
                            }
                            
                            int main() {
                                int n,t;
                                std::cin>>t;
                                while (t--) {
                                    std::cin>>n;
                                    for (int i=1;i<=n;i++) {
                                        std::cin>>a[i];
                                    }
                                    std::sort(a+1,a+1+n);
                                    for (int i=1;i<n;i++) {
                                        b[i]=a[i+1]-a[i];
                                    }
                                    int t=0;
                                    for (int i=1;i<n;i++) {
                                        t=gcd(b[i],t);
                                    }
                                    std::cout<<res(t)<<'\n';
                                }
                                return 0;
                            }

Problem F 人见人爱$ \boldsymbol{A^B} \: \mathbf{mod} \: \boldsymbol{C} $

[题目描述]

计算A的B次幂对C取模($ A^B \: \mathrm{mod} \: C $)并且1≤A，B,C≤32768。输入三个整数A,B,C，输入文件每一行依次三个整数A,B,C用空格隔开，每行输出一个运算结果。

[样例输入]

                        
                            3 4 5
                            10 10 100
                            45 67 89
                            5 55555555 3
                            6 12345678 123

[样例输出]

                        
                            1
                            0
                            45
                            2
                            90

[解答]

经典的快速幂。 

                        
                            #include<iostream>
                            int main() {
                                int a,b,c,j;
                                while (scanf("%d %d %d",&a,&b,&c)!=EOF) {
                                    j=1;
                                    while (b>0) {
                                        if (b%2!=0) j=(j*a)%c;
                                        b=b/2;
                                        a=(a*a)%c;
                                    }
                                    std::cout<<j<<'\n';
                                }
                            }
