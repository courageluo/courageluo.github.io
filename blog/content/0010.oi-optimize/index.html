OI中的骗分及禁赛级优化

以下介绍一些在OI测试中，无法通过降低算法时间复杂度（想不出更优算法），或者根本做不出来的情况下，降低运行时间或得到一定分数的方法。

骗分

  1. 样例及无解情况
    直接输出样例及无解情况的答案。 
                                
                                int main() {
                                    int n;
                                    cin>>n;
                                    if (n==样例输入) {
                                        cout<<样例输出;
                                    } else {
                                        cout<<无解输出;
                                    }
                                    return 0;
                                }
  2. 打表
    如果能够通过暴力求解或者能够直接求出问题答案，并且数据都比较小的情况下，可以先写暴力程序，然后将输出写入另一个程序中直接读取对应答案。
    例如 NOIP2003 栈 这个题，由于n≤18，数据极小，所以可以直接 
                                
                                int main() {
                                    int ans[18]={1,2,5,14,42 ... 129644790,477638700};
                                    int n;
                                    cin>>n;
                                    cout<<ans[n-1];
                                    return 0;
                                }

优化

适用于缩短暴力算法的运行时间，尽可能少TLE。

  1. 快读快写
    通过getchar()、putchar()等函数尽可能地降低输入输出时间。 
                                
                                int read() {
                                    int x=0,f=1;
                                    char ch=getchar();
                                    while (ch<'0'||ch>'9') {
                                        if (ch=='-') f=-1;
                                        ch=getchar();
                                    }
                                    while (ch>='0'&&ch<='9') {
                                        x=x*10+ch-'0',ch=getchar();
                                    }
                                    return x*f;
                                }
                            
                        
                                
                                void write(int x) {
                                    if (x<0) {
                                        putchar('-'),x=-x;
                                    }
                                    if (x>9) {
                                        write(x/10);
                                    }
                                    putchar(x%10+'0');
                                    return;
                                }
  2. 使用inline、register等关键字
    在函数声明前加上inline关键字，编译器会尽可能地将函数展开，从而减少运行时间。 
                                
                                inline int max(int a,int b) {
                                    return a>b?a:b;
                                }
    在一些变量前加上register关键字，可以使变量尽可能地存储在寄存器之中，增加变量的读取速度，从而减少运行时间。一般写在循环变量之中，对普通的变量用处不大，而且它已经在C++17后被弃用。 
                                
                                for (register int i=1;i<=n;i++) {
                                    for (register int j=1;j<=m;j++) {
                                        // 循环内容
                                    }
                                }
  3. 使用位运算
    位运算的效率通常更高，因此使用位运算可以减少运行时间。 
                                
                                x<<=n      // x*=2ⁿ
                                x>>=n      // x/=2ⁿ
                                x&1        // x%2==1
                                -x&1       // x%2==0
                                y-x>>31    // x>y (int x,y)
                                y-x>>63    // x>y (long long x,y)
                                ~x+1       // -x
                                x^=y^=x^=y // swap(x,y)
  4. 展开循环
    手动展开一些循环，可以刺激CPU多核的运算，从而减少运行时间。建议展开数量为4到8个。 
                                
                                for (int i=1;i<=n;i+=8) {
                                    a[i]++; a[i+1]++; a[i+2]++; a[i+3]++;
                                    a[i+4]++; a[i+5]++; a[i+6]++; a[i+7]++;
                                }
  5. 增强内存访问连续性
    在使用高维数组时，应该尽量使最后一维变化连续，可以减少内存访问时间。 
                                
                                // 建议 b[k][j]的访问相对更加连续
                                for (int i=1;i<=n;i++)
                                    for (int k=1;k<=p;k++)
                                        for (int j=1;j<=m;j++)
                                            ans[i][j]+=a[i][k]*b[k][j];

                                // 不建议 b[k][j]的访问相对不那么连续
                                for (int i=1;i<=n;i++)
                                    for (int j=1;j<=m;j++)
                                        for (int k=1;k<=p;k++)
                                            ans[i][j]+=a[i][k]*b[k][j];
  6. 其它优化
    无法使用快读快写的情况下，可以使用 
                                
                                ios::sync_with_stdio(false);
                                cin.tie(0);
                                cout.tie(0);
    来加速cin和cout的输入输出速度。
    输出换行符时，使用'\n'而不是endl，可以减少刷新操作，提升运行效率。

禁赛级优化

警告：以下方法不建议在比赛中使用，否则后果自负。

  1. 火车头优化
    将一系列GCC编译器的优化指令放在程序的预处理中，可以极大地减少运行时间。 
                                
                                #pragma GCC optimize(3)
                                #pragma GCC target("avx")
                                #pragma GCC optimize("Ofast")
                                #pragma GCC optimize("inline")
                                #pragma GCC optimize("-fgcse")
                                #pragma GCC optimize("-fgcse-lm")
                                #pragma GCC optimize("-fipa-sra")
                                #pragma GCC optimize("-ftree-pre")
                                #pragma GCC optimize("-ftree-vrp")
                                #pragma GCC optimize("-fpeephole2")
                                #pragma GCC optimize("-ffast-math")
                                #pragma GCC optimize("-fsched-spec")
                                #pragma GCC optimize("unroll-loops")
                                #pragma GCC optimize("-falign-jumps")
                                #pragma GCC optimize("-falign-loops")
                                #pragma GCC optimize("-falign-labels")
                                #pragma GCC optimize("-fdevirtualize")
                                #pragma GCC optimize("-fcaller-saves")
                                #pragma GCC optimize("-fcrossjumping")
                                #pragma GCC optimize("-fthread-jumps")
                                #pragma GCC optimize("-funroll-loops")
                                #pragma GCC optimize("-fwhole-program")
                                #pragma GCC optimize("-freorder-blocks")
                                #pragma GCC optimize("-fschedule-insns")
                                #pragma GCC optimize("inline-functions")
                                #pragma GCC optimize("-ftree-tail-merge")
                                #pragma GCC optimize("-fschedule-insns2")
                                #pragma GCC optimize("-fstrict-aliasing")
                                #pragma GCC optimize("-fstrict-overflow")
                                #pragma GCC optimize("-falign-functions")
                                #pragma GCC optimize("-fcse-skip-blocks")
                                #pragma GCC optimize("-fcse-follow-jumps")
                                #pragma GCC optimize("-fsched-interblock")
                                #pragma GCC optimize("-fpartial-inlining")
                                #pragma GCC optimize("no-stack-protector")

                                #pragma GCC optimize("-findirect-inlining")
                                #pragma GCC optimize("-fhoist-adjacent-loads")
                                #pragma GCC optimize("-frerun-cse-after-loop")
                                #pragma GCC optimize("inline-small-functions")
                                #pragma GCC optimize("-finline-small-functions")
                                #pragma GCC optimize("-ftree-switch-conversion")
                                #pragma GCC optimize("-foptimize-sibling-calls")
                                #pragma GCC optimize("-fexpensive-optimizations")

                                #pragma GCC optimize("inline-functions-called-once")
                                #pragma GCC optimize("-fdelete-null-pointer-checks")
                                #pragma GCC optimize(2)
    不过一般OJ都自带了O2优化。
  2. 内嵌汇编
    汇编语言执行效率在计算机中是非常高的，所以可以内嵌汇编来提高代码的执行效率。 
                                
                                #include<iostream>
                                using namespace std;

                                int main() {
                                    int a,b,ans;
                                    cin>>a>>b;
                                    __asm__ (
                                        "movl %1,%%eax\n\t"
                                        "addl %2,%%eax\n\t"
                                        "movl %%eax,%0"
                                        : "=r"(ans)
                                        : "r"(a),"r"(b)
                                    ); 
                                    cout<<ans<<endl;
                                    return 0;
                                }
  3. 使用AVX指令集
    指令集是存储在CPU内部的一组指令，用于控制CPU的运行，它可以大大提高运行效率。AVX指令集是Intel公司推出的一种指令集。 
                                
                                #include<iostream>
                                #include<immintrin.h>
                                using namespace std;

                                int main() {
                                    float a,b,ans;
                                    cin>>a>>b;
                                    __m256 va=_mm256_set_ps(0,0,0,0,0,0,0,a);
                                    __m256 vb=_mm256_set_ps(0,0,0,0,0,0,0,b);
                                    __m256 vans=_mm256_add_ps(va,vb);
                                    ans=_mm256_cvtss_f32(vans);
                                    cout<<ans<<endl;
                                    return 0;
                                }
  4. 使用SIMD指令
                                
                                #include<iostream>
                                #include<immintrin.h>
                                using namespace std;

                                int main() {
                                    float a,b;
                                    cin>>a>>b;
                                    __m128 va=_mm_set_ss(a);
                                    __m128 vb=_mm_set_ss(b);
                                    __m128 vans=_mm_add_ss(va,vb);
                                    cout<<_mm_cvtss_f32(vans)<<endl;
                                    return 0;
                                }
  5. 多线程并行化 
                                
                                #include<iostream>
                                #include<future>
                                #include<thread>
                                using namespace std;

                                float add(float a,float b) {
                                    return a+b;
                                }

                                int main() {
                                    float a,b;
                                    cin>>a>>b;
                                    auto future=async(launch::async,add,a,b);
                                    cout<<future.get()<<endl;
                                    return 0;
                                }
  6. 内存对齐和缓存优化 
                                
                                #include<iostream>
                                #include<cstdlib>
                                using namespace std;

                                int main() {
                                    alignas(64) float a,b;
                                    cin>>a>>b;
                                    cout<<a+b<<endl;
                                    return 0;
                                }
  7. 减少I/O操作 
                                
                                #include<iostream>
                                #include<string>
                                using namespace std;

                                int main() {
                                    string input;
                                    getline(cin,input);
                                    size_t space=input.find(' ');
                                    float a=stof(input.substr(0,space));
                                    float b=stof(input.substr(space+1));
                                    cout<<a+b<<endl;
                                    return 0;
                                }
  8. GPU加速
    这里使用CUDA进行演示，需要NVIDIA显卡和相关驱动及软件，也可以使用OpenCL等其它库。 
                                
                                #include<iostream>
                                #include<cuda_runtime.h>
                                using namespace std;

                                __global__ void add(float *a,float *b,float *result) {
                                    *result=*a+*b;
                                }

                                int main() {
                                    float a,b,result;
                                    cin>>a>>b;
                                    float *d_a,*d_b,*d_result;
                                    cudaMalloc(&d_a,sizeof(float));
                                    cudaMalloc(&d_b,sizeof(float));
                                    cudaMalloc(&d_result,sizeof(float));
                                    cudaMemcpy(d_a,&a,sizeof(float),cudaMemcpyHostToDevice);
                                    cudaMemcpy(d_b,&b,sizeof(float),cudaMemcpyHostToDevice);
                                    add<<<1,1>>>(d_a,d_b,d_result);
                                    cudaMemcpy(&result,d_result,sizeof(float),cudaMemcpyDeviceToHost);
                                    cout<<result<<endl;
                                    cudaFree(d_a); cudaFree(d_b); cudaFree(d_result);
                                    return 0;
                                }

上面演示的都是处理a+b问题，略显小题大做，只有在真正处理大量数据和运算时才会有明显的优化。
